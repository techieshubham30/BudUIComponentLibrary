Avatar

Avatar Component can be used for graphical representation of a person through a profile picture or image.

Example of Image Avatar

Avatar is available in 4 diffrent sizes. You need to include class avatar and for diffrent sizes you can add classes avatar-lg-size, avatar-md-size, avatar-sml-size and avatar-xsml-size.
